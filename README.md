# Project-example SPA to SSR

## Work with api


```bash
# move to api
cd api
```
```bash
# install packages
npm run install
```
```bash
# start api
npm run start
```

## Work with SPA
Before launch SPA need start api

```bash
# move to spa
cd spa
```
```bash
# install packages
npm run install
```
```bash
# start spa
npm run dev
```
