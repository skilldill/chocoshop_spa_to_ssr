# Project-example SPA to SSR

## Work with API


```bash
# move to api
cd api
```
```bash
# install packages
npm install
```
```bash
# start api
npm run start
```

## Work with SPA
Before launch SPA need start api

```bash
# move to spa
cd spa
```
```bash
# install packages
npm install
```
```bash
# start spa
npm run dev
```

## Work with SSR
Before launch SSR need start api

```bash
# move to ssr
cd ssr
```
```bash
# install packages
npm install
```
```bash
# start ssr
npm run start
```
